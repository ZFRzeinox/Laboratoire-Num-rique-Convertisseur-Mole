# Laboratoire Numérique Convertisseur & Mole

Un programme Python interactif conçu pour aider en laboratoire à convertir des volumes et calculer des quantités de matière (moles) rapidement en TP.

## Fonctionnalités
- Conversion instantanée de millilitres ($mL$) en Litres ($L$).
- Calcul automatique du nombre de moles ($n$) à partir de la masse ($m$) et de la masse molaire ($M$).
- Gestion des erreurs si l'utilisateur se trompe dans le menu.

## Compétences Python développées
- Gestion des types de variables (`int`, `float`).
- Conditions et structures logiques (`if`, `elif`, `else`).
- Affichage dynamique avec les `f-strings`.
